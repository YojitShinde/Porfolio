Placeable

The primary objective of this web-app is to simplify the placement process for students, placement officers, and companies, by providing a platform that makes it easy to manage job postings, applications and student queries.

This project was made using Angular, Python (Django) and PostgreSQL. The source-code can be found here.



Desklet

This web-app tries to simplify the management and the tracking of Purchase Orders (POs), Sales Orders (SOs) and Materials Received Notes (MRNs) for an enterprise. The web-app is equipped with an automated email-generation for the required PO, SO or MRN.

This project was made using Angular, Java (Spring) and MySQL. The source-code can be found here.



Doable

A Simple To-Do-List Application for Android Devices. The app consists of a handy 'Read List' feature to assist the visually impared. Additionally, users can share their list with the 'Share List' feature.

This project was made using Android Studio and Java. The source-code can be found here.
